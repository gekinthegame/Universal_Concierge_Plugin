⚙ Deploy settings
Tokens stay on this device only (0600 file) and are used solely to deploy to the host you choose — never shared, never sent anywhere but that platform's own API. IPFS needs no token; these Web2 hosts are optional opt-ins.

Approved peers

Requests
A message from someone you haven't approved waits here as a request — a public username is never enough to reach you. Accept to start the thread, or decline to block.
Network discovery map
Your node at the center; the peers it finds on the libp2p network around it — mDNS on your LAN, the public DHT, and rendezvous points. Each brain is a node: a solid link is a live connection, a dashed link is discovered-but-not-connected. Opening this tab brings your node online so discovery runs.
